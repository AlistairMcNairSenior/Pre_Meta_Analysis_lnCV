Files associated with the analyses and simulations in Senior et al. Bias in meta-analysis of response ratios is reduced by preliminary meta-analysis of the variance

The files are broadly seperated in to two parts; the simulations and the worked examples

Simulations
Each simulation is designed to run on an array on a HPC. 
Running the script 1.Simulation_1.R will run 100 replicate simulations for simulation 1, with the output put in to the directory Output_Simulation_1. 
For the publication 1000 instances of that script were run.
Mulitple instances can be aggregated and processed using the next script in the sequence (e.g., 2.Aggregate_Simulation_1.R).

Worked Examples 
Each worked example is in a seperarte script with the data in a corresponding .csv file.
